❓ Help — Comic Volume Creator

What This Does

Comic Volume Creator scans a folder for individual comic files (.cbz, .cbr) and merges them into organized volumes. It's designed for large comic collections where you want to group individual issues (e.g., #001–#010) into a single merged archive (v01.cbz).

Quick Start

  1. Enter source path: Set the folder containing your comics (top bar)
  2. Scan folders: Click the blue "🔍 Scan Folders" button
  3. Review results: See all series found, grouped by file count
  4. Create volumes: Click "✨ Create" on a row, or select multiple and bulk create

Flag Legend

Each series is marked with a flag indicating how many individual files it has:

  • Y = 4–6 files (ready to merge)
  • S = <4 files (small series)
  • M = >6 files (large volume)

Filters

  • Status Filter: Ready (Y/M) | CBZ exists | Redundant | Single file
  • Flag Filter: Y | S | M | All flags
  • Search: Type series name to find matches

Single Volume Creation

  1. Click the "✨ Create" button on a row
  2. Review the proposed output name and command
  3. Edit the name if needed (e.g., change v01 to custom name)
  4. Click "✨ Create" to confirm and merge

Process: extracts all files → zips them together → moves to folder → deletes originals

Split Large Volumes (Flag: M)

For series with more than 6 files, a "Split" button appears. Use it to divide into multiple volumes:

  1. Click "Split" on the row
  2. Check files for v01, then assign volume name
  3. Check remaining files, change volume to v02, assign
  4. Or click "💡 Suggest Split" for auto-balanced split
  5. Review preview, then click "✨ Create Volumes"

Bulk Operations

  1. Check checkboxes next to rows (left side)
  2. Use "☑ Select All" / "☐ Deselect All" shortcuts
  3. Click "📋 Dry Run" to preview what would happen
  4. Click "✨ Create Selected CBZs" to merge all at once

Console Output

Click the "🖥 Console" button (bottom-right) to see detailed merge progress:

  • ✔ Green = successful steps
  • ✖ Red = errors
  • ℹ Blue = information
  • ⚠ Orange = warnings

Settings & Themes

Click the "⚙ Settings" button (top-right) to:

  • Choose a colour theme (10 themes available)
  • Adjust font size (small, medium, large)
  • Change table density (compact, normal, relaxed)

Settings are saved to your browser.

Keyboard Shortcuts

Escape — Close dialogs
Enter (in path field) — Start scan
Enter (in confirm dialog) — Confirm creation

Troubleshooting

  • No results after scan? Ensure files are named like "Series 001.cbr" (numbered issues)
  • Server not found? Run: python3 comic_volume_creator_server.py
  • Files not deleted after merge? Check console for errors; originals are only deleted on success
