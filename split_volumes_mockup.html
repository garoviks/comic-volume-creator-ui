How It Works

1. Checkbox Selection

User checks files 1–10. The "Assign Volume" panel shows the count and has a text input for the volume name.

2. Smart Volume Detection

Server scans the folder. Finds Largo Winch v01 (2008).cbz exists. When user checks files, the input field auto-populates with the next volume number: v02.

3. Preview + Confirmation

Right panel shows: existing volumes (read-only, greyed out), and new volumes being created with file counts. User can select more files and assign them to v03, etc.

4. Dry Run or Create

Click "Dry Run" to preview without executing. Click "✨ Create Selected Volumes" to merge. Server runs /api/create for each volume sequentially.

5. (Phase 2) Auto-Suggest Button

"💡 Suggest Split" proposes balanced splits: 20 files → 2×10 each, or 3×7+6+7, depending on logic. User can accept or override with manual selection.
